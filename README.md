# TrailerFlix

This is a site to view film trailers with a Netflix-like UI.

Built using Next JS and Tailwind CSS.

## Planned updates

- Search functionality
- Ability to view more information about a film, such as actors, box office etc...

## The Move Database

This product uses the [TMDB](https://www.themoviedb.org/) API but is not endorsed or certified by TMDB.
